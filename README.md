# Alaa Gamal — Professional Portfolio

A modern, responsive portfolio website showcasing data analytics and engineering projects, including Power BI dashboards, Python exploratory data analysis, and SQL query sets.

## Tech Stack

- **React 19** — UI library
- **TypeScript** — Programming language
- **Vite 7** — Build tool and dev server
- **Tailwind CSS 4** — Utility-first styling
- **Radix UI** — Accessible UI primitives
- **TanStack React Query** — Data fetching and caching
- **Framer Motion** — Smooth animations & transitions
- **Wouter** — Lightweight router

## Getting Started

### Prerequisites

Make sure you have Node.js (v18+) and `pnpm` installed.

### Installation

Install dependencies:

```bash
pnpm install
```

### Running Locally

To run the development server:

```bash
pnpm dev
```

The app will start on [http://localhost:5173](http://localhost:5173).

### Building for Production

To build the application for deployment:

```bash
pnpm build
```

The static files will be generated in the `/dist` directory.
